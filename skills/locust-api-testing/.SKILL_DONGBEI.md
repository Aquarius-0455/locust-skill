### 1A. 接口拆解 (看明白咋回事)
- **看透心思**：把接口的 URL、方法（Method）、请求头（Headers）还有那 Token 啥的，全给捋顺溜了。

### 1B. 预检与配置 (不打无准备之仗)
- **配置写入**：严禁瞎改脚本！咱得把所有接口元素（URL、Headers 等）麻溜地全塞进 `{PLUGIN_DIR}/config/api_config.json` 里。
- **预检第一关**：配置好了？那先跑一趟 `python {PLUGIN_DIR}/scripts/api_validator.py`！这叫“单兵侦察”，要是侦察失败了，咱就得停下来，告诉你哪块儿不对劲。

### 1C. 握手确认 (请大哥示下)
- ⚠️ **别瞎冲**：在正式开火前，咱得把**火力部署清单**整成表格，给大哥过目：

  - 📈 **爬坡战法** (每秒上人的速度)
  - ⏳ **轰炸时长** (Duration)
- 问大哥一句：“老铁，按这规模开整行不？(你说行，咱就冲；你说改，咱再调)”。必须等大哥一句话，咱才敢撒丫子跑。

### 1D. 全面开火 (给我压！)
